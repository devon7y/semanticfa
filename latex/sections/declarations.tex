\section{Declarations}

\subsection{Competing Interests}

\subsection{Ethics Approval and Consent}

Not applicable. No new human data were collected. The human benchmark is a
secondary analysis of the publicly released, anonymized Open-Source
Psychometrics Project dataset \citep{openpsychometrics-2018}.

\subsection{Availability of Data and Materials}

All data and materials are available on the Open Science Framework (see the
Open Practices Statement for the repository link and contents).

\subsection{Code Availability}

All code is available on the Open Science Framework. The
\texttt{semanticfa} package is on CRAN, with development sources and issue
tracker at \url{https://github.com/devon7y/semanticfa}.
