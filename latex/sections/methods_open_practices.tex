\subsection{Open Practices}

A self-contained reproduction archive, to be hosted on the Open Science Framework at \url{https://osf.io/XXXXX} [OSF link to be inserted], regenerates every number, table, and figure in the demonstration from raw inputs. It contains the master script (\texttt{reproduce.R}); a full-run console transcript (\texttt{reproduce.Rout}) for checking every command against its real output; the precomputed Qwen3-Embedding-8B item and auxiliary embedding archives with the auxiliary-vector generation script; results as an \texttt{.rds} object and CSV tables including the encoding comparison, congruence matrix, and both loading matrices; and the regenerated figures. The human response file is not redistributed where licensing forbids it; the script documents the public-release download \citep{openpsychometrics-2018} and accepts its location via an environment variable. Only two steps require Python or network access---the NLI matrix and the live-embedding demonstrations, which fetch models from the Hugging Face Hub on first run; all factor analyses, retention methods, diagnostics, congruence metrics, refinement analyses, and figures are pure R and run offline.

The reported run used R 4.5.0 \citep{rcoreteam-2025} with semanticfa 0.1.0 \citep{yanitski-westbury-2026}, psych 2.6.5 \citep{revelle-2026}, EGAnet 2.4.1 \citep{golino-christensen-2026}, GPArotation 2026.4-1, Rtsne 0.17, uwot 0.2.4, and reticulate 1.43.0, with seed 42. The package is on CRAN, with development sources and issue tracker at \url{https://github.com/devon7y/semanticfa}; a vignette walks every exported function through a second, independent worked example.
